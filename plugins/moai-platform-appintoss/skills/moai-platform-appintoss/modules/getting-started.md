# 시작하기 (Getting Started)

앱인토스 미니앱 개발을 시작하기 위한 가이드입니다.

---

## 플랫폼 개요

앱인토스(Apps in Toss)는 파트너사가 개발한 서비스를 토스 앱 내부에서 App-in-App 형태로 제공하는 플랫폼입니다.

### 플랫폼 특징

**대규모 사용자 접근:**
- 3,000만 토스 사용자에게 서비스 노출
- 토스 앱 내 자연스러운 진입점 제공
- 별도 앱 설치 없이 즉시 사용 가능

**통합 기능:**
- 토스 로그인: OAuth2 기반 간편 인증
- 토스페이: mTLS 인증 기반 안전한 결제
- 푸시 알림: 프로모션/기능성 메시지
- 광고: 전면 광고, 보상형 광고
- 마케팅: 세그먼트 타겟팅, 토스 포인트 프로모션

**개발 이점:**
- SDK 제공: WebView 및 React Native 기반 SDK
- 간편한 배포: 빌드 결과물 업로드 후 검수, 출시
- 네이티브 개발 불필요: 복잡한 네이티브 개발 없이 앱 제작

## 개발 옵션

앱인토스는 두 가지 개발 방식을 지원합니다.

### Option A: WebView 개발

**Vite + React + TypeScript** 기반으로 간단한 앱에 적합합니다.

**장점:**
- 빠른 개발 속도
- 웹 개발 경험 활용
- 간단한 설정

**단점:**
- 네이티브 기능 제한
- 성능 최적화 한계

**적합한 앱:**
- 정보 제공 서비스
- 단순 폼 기반 앱
- 컨텐츠 중심 앱

### Option B: React Native (Granite)

**Granite 프레임워크**를 사용하며 복잡한 앱에 적합합니다.

**장점:**
- 네이티브 성능
- 풍부한 애니메이션
- 복잡한 UI 구현 가능

**단점:**
- 학습 곡선
- 설정 복잡도

**적합한 앱:**
- 게임
- 복잡한 인터랙션
- 성능 중요 앱

## 콘솔 워크스페이스 설정

### Step 1: 앱인토스 콘솔 접근

개발자센터에 접속하여 계정을 생성합니다.

URL: https://developers-apps-in-toss.toss.im

### Step 2: 미니앱 생성

1. 콘솔 로그인
2. "새 미니앱 만들기" 클릭
3. 앱 정보 입력:
   - 앱 이름 (영문, 하이픈 사용)
   - 카테고리 선택 (게임/비게임)
   - 서비스 설명

### Step 3: 개발 환경 설정

1. SDK 타입 선택 (WebView / React Native)
2. 앱 식별자 확인
3. mTLS 인증서 발급 (로그인/결제 사용 시)

## 사업자 등록

앱인토스 서비스를 정식 출시하려면 사업자 등록이 필요합니다.

### 필요 서류

**법인 사업자:**
- 사업자등록증
- 법인등기부등본
- 인감증명서

**개인 사업자:**
- 사업자등록증
- 신분증 사본

### 등록 절차

1. 콘솔에서 "사업자 등록" 메뉴 접근
2. 사업자 정보 입력
3. 서류 업로드
4. 검토 (영업일 기준 2-3일)
5. 승인 완료

---

## 개발 흐름 요약

```
1. 콘솔에서 미니앱 생성
       ↓
2. 개발 환경 선택 (WebView / RN)
       ↓
3. SDK 설치 및 초기화
       ↓
4. 로컬 개발 및 테스트
       ↓
5. 샌드박스 앱에서 테스트
       ↓
6. 토스 앱에서 QR 테스트
       ↓
7. 번들 빌드 및 업로드
       ↓
8. 검수 (운영/디자인/기능/보안)
       ↓
9. 출시
```

## 주의사항

### 개발 시 주의사항

1. **TDS 필수 사용**: 토스 디자인 시스템(TDS) 컴포넌트 사용 필수
2. **라이트 모드 전용**: 다크 모드 미지원 (라이트 모드만 구현)
3. **핀치 줌 비활성화**: 줌 기능 비활성화 필수
4. **접근성 준수**: 대비, 터치 타겟 등 접근성 요구사항 충족

### 정책 주의사항

1. **토스 로그인만 사용**: 서드파티 로그인(카카오, 네이버 등) 금지
2. **토스페이만 사용**: 다른 결제 수단 금지
3. **다크패턴 금지**: 사용자 기만적 UI 패턴 금지
4. **광고 정책 준수**: 예상치 못한 광고 노출 금지

### 기술 제한사항

1. **번들 크기**: 최대 100MB (비압축 기준)
2. **앱 카테고리**: 게임/비게임 선택 후 변경 불가
3. **mTLS 인증서**: 유효기간 390일, 갱신 필수
